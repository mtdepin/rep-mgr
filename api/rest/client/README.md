## Install

You can import `github.com/mtdepin/rep-mgr/api/rest/client` in your code.

The code can be downloaded and tested with:

```
$ git clone https://github.com/mtdepin/rep-mgr.git
$ cd mtdepin/rep-mgr/rest/api/client
$ go test -v
```
